# Minimal Resume Example

This example demonstrates the absolute minimum required fields for a valid resume configuration.

## What's Included

- **Profile**: Name, email, and introduction (minimum required fields)
- **Experience**: One entry with company, position, and dates
- **Education**: One entry with institution, degree, and dates

## What's Optional

Everything else! You can add:
- Phone, location, website to profile
- Skills, languages, hobbies sections
- Social media links
- Detailed descriptions and task lists
- Additional experience and education entries

## Build This Example

```bash
node src/cli.js build -c examples/minimal/resume.yaml -o dist-minimal
```

## Starting Point

This is a great starting point if you want to:
1. Test the tool quickly
2. Build up your resume incrementally
3. Keep your resume simple and focused

Copy this to your resumes directory and expand as needed:

```bash
cp examples/minimal/resume.yaml resumes/resume.yaml
```
